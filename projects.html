Nashville Software School Projects

Below are a collection of class projects I completed during the front-end engineering program at Nashville Software School. My earlier projects feature code written in basic HTML & CSS; the rest were developed using Javascript. My capstone, Lucid, was written in React. My proudest accomplishment was developing high-fidelity prototypes of all these projects in Figma. Please feel free to visit my Github page to view more projects.


Capstone: Lucid

Lucid was my front-end capstone. It's a dream tracking app that can interpret several dream components simultaneously. This project was written in React and features a pretty extensive mock-up design.

Pinterest

This mock is a simplified version of the popular website Pinterest, which is a visual discovery engine for finding ideas like recipes, home, and style inspiration. It was made using Javascript and SASS.

Bear Watcher

Bear Watcher is a mock app whoes mainly used by rangers, scientists and bear enthusiasts to track and research bears in the Katmai National Park. This app demonstrates data collection linked with Javascript to produce bear profiles.
